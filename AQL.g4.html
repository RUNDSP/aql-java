  • create Top

    CREATE INDEX ON [.] () NUMERIC|STRING

    Text notation:

    create : CREATE ((LIST | MAPVALUES | MAPKEYS)? INDEX index_name ON nameSet {definitions.add(VariableDefinition.WRITE_POLICY);} '(' binName=bin ')' iType=(NUMERIC | STRING) {definitions.add(VariableDefinition.INDEX_TASK);} | USER user PASSWORD password (ROLE role | ROLES roles*) {definitions.add(VariableDefinition.ADMIN_POLICY);} | ROLE role (PRIVILEGE | PRIVILEGES) privilege* {definitions.add(VariableDefinition.ADMIN_POLICY);} ) ;

    Visual notation:

  • drop Top

    DROP INDEX [.] DROP MODULE modulename.extension DROP SET [.] DROP USER

    Text notation:

    drop : DROP ( INDEX nameSet index_name | MODULE moduleName | SET nameSet | USER user {definitions.add(VariableDefinition.ADMIN_POLICY);} | ROLE role {definitions.add(VariableDefinition.ADMIN_POLICY);} ) ;

    Visual notation:

  • repair Top

    REPAIR INDEX ON [.]

    Text notation:

    repair : REPAIR INDEX index_name ON nameSet ;

    Visual notation:

  • grant Top

    Text notation:

    grant : GRANT (ROLES roles | ROLE role) TO user | (PRIVILEGE privilege | PRIVILEGES privilege+) TO role {definitions.add(VariableDefinition.ADMIN_POLICY);} ;

    Visual notation:

  • revoke Top

    Text notation:

    revoke : REVOKE (ROLES roles | ROLE role ) FROM user | (PRIVILEGE privilege | PRIVILEGES privilege+) FROM role {definitions.add(VariableDefinition.ADMIN_POLICY);} ;

    Visual notation:

  • privilege Top

    Text notation:

    privilege : ('read' { $priv.code = PrivilegeCode.READ; } | 'read-write' ('.' nameSet)? { $priv.code = PrivilegeCode.READ_WRITE; } | 'read-write-udf' ('.' nameSet)? { $priv.code = PrivilegeCode.READ_WRITE_UDF; } ) ('.' nameSet { $priv.namespace = $nameSet.namespaceName; $priv.setName = $nameSet.setName; })? | 'sys-admin' { $priv.code = PrivilegeCode.READ_WRITE_UDF; } | 'user-admin' { $priv.code = PrivilegeCode.READ_WRITE_UDF; } ;

  • operate Top

    Text notation:

    operate : OPERATE operateFunction (',' operateFunction)* ON nameSet WHERE primaryKeyPredicate (AND generationPredicate)? { definitions.add(VariableDefinition.WRITE_POLICY); definitions.add(VariableDefinition.READ_POLICY); definitions.add(VariableDefinition.RECORD); } ;

    Visual notation:

  • insert Top

    INSERT INTO namespace[.setname] (PK,binnames,,,) VALUES ('pk',nnn,'xxx',,)

    Text notation:

    insert : INSERT INTO nameSet '(' PK ',' binNameList ')' VALUES '(' primaryKey[$statement::nameSpace, $statement::setName] ',' valueList ')' { definitions.add(VariableDefinition.WRITE_POLICY); } ;

    Visual notation:

  • update Top

    UPDATE [.] SET = , ... WHERE PK = [AND generation = ]

    Text notation:

    update : UPDATE nameSet SET updateList WHERE primaryKeyPredicate (AND generationPredicate)? { definitions.add(VariableDefinition.WRITE_POLICY); } ;

    Visual notation:

  • updateList Top

    Text notation:

  • delete Top

    DELETE FROM namespace[.setname] WHERE PK = 'x'

    Text notation:

    Visual notation:

  • select Top

    SELECT *|bin,,, FROM namespace[.setname] SELECT *|bin,,, FROM namespace[.setname] ORDER BY xxx SELECT *|bin,,, FROM namespace[.setname] WHERE bin = nnn SELECT *|bin,,, FROM namespace[.setname] WHERE bin = 'xxx' SELECT *|bin,,, FROM namespace[.setname] WHERE bin BETWEEN nnn AND mmm SELECT *|bin,,, FROM namespace[.setname] WHERE PK = 'x' Note:SELECT...WHERE bin ... needs an INDEX on bin. Use CREATE INDEX first.

    Text notation:

    select : SELECT ( STAR { definitions.add(VariableDefinition.RECORD); } | binNameList { definitions.add(VariableDefinition.RECORD); } | moduleFunction ('(' (valueList)? ')')? { definitions.add(VariableDefinition.RESULT_SET); } ) (IN collectionType)? FROM nameSet where? { definitions.add(VariableDefinition.QUERY_POLICY); definitions.add(VariableDefinition.READ_POLICY); definitions.add(VariableDefinition.SCAN_POLICY); definitions.add(VariableDefinition.STMT); } ;

    Visual notation:

  • aggregate Top

    AGGREGATE pkgname.funcname(arg1,arg2,,) ON namespace[.setname] WHERE bin = nnn AGGREGATE pkgname.funcname(arg1,arg2,,) ON namespace[.setname] WHERE bin BETWEEN nnn AND mmm

    Text notation:

    aggregate : AGGREGATE moduleFunction ('(' (valueList)? ')')? ON nameSet (IN collectionType)? (where)? { definitions.add(VariableDefinition.QUERY_POLICY); definitions.add(VariableDefinition.RESULT_SET); definitions.add(VariableDefinition.STMT); } ;
